# 🌿 GreenTruth Auditor

## 🚀 Overview

GreenTruth Auditor is an AI-powered tool that detects greenwashing in product descriptions by analyzing sustainability claims using NLP and Machine Learning.

---

## ❗ Problem

Brands often use vague terms like “eco-friendly” without proof, misleading consumers.

---

## 💡 Solution

Our system:

* Classifies claims as **Fluff vs Evidence**
* Detects buzzwords
* Verifies certifications
* Generates a **Greenwash Intelligence Score**

---

## 🧠 Tech Stack

* Streamlit (UI)
* Transformers (BART MNLI, FLAN-T5)
* NLTK
* Python

---

## ⚙️ Features

* Text / URL input
* Explainable AI results
* Certification verification (RAG)
* AI-powered claim rewriting

---

## 📊 How it works

1. Input product description
2. Split into sentences
3. Classify each claim
4. Detect buzzwords & certifications
5. Generate score
